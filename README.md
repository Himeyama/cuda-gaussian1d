## Installation

### Dependency
```bash
brew install himeyama/homebrew-lib/libcuda-gaussian1d
```

### Gem or Gemfile
> Gemfile
```rb
gem 'cuda-gaussian1d'
```

and

```bash
bundle
```

> Gem
```bash
gem install cuda-gaussian1d
```
